# 📈 Stock Portfolio Tracker

## 📖 Description

This is a simple Stock Portfolio Tracker developed in Python as part of the CodeAlpha Python Programming Internship. The application allows users to enter stock names and quantities, calculates the total investment value using predefined stock prices, displays the portfolio details, and saves the investment summary to a text file.

## ✨ Features

- Calculates the total investment value based on stock prices and quantities.<br>
- Supports multiple stock entries.<br>
- Uses a predefined dictionary to store stock prices.<br>
- Displays the portfolio summary in the console.<br>
- Saves the portfolio details to a portfolio.txt file.<br>
- Validates stock names before processing.
